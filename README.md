# AI-Powered Food–Drug Interaction System

## Overview
A full-stack AI-powered web application that analyzes potential food and drug interactions to provide medication safety insights.

## Tech Stack
- Frontend: React.js (Vite)
- Backend: Node.js, Express.js
- Database: MongoDB
- AI: AI-based analysis API
- Deployment: Vercel (Frontend), Render (Backend)

## Features
- User-friendly interface for medication input
- AI-based food–drug interaction analysis
- RESTful APIs for backend communication
- Secure and scalable architecture
